> آخرین به‌روزرسانی: YYYY-MM-DD

## نحوه استفاده

| نقش | نحوه استفاده |
|-----|----------------|
| AI (Cursor, Claude, …) | یک فایل را باز کنید؛ تمام سکشن‌ها context یکپارچه می‌دهند |
| طراحی | `UI Ideas`, `Components`, `00-brand` |
| فرانت | `Sections`, `Components`, `18-technical` |
| محتوا | `Goal`, `SEO Keywords`, `brand-voice` |
| مارکتینگ | `Conversion Strategy`, `17-marketing` |

## وضعیت‌ها

- **پیش‌نویس**: ساختار آماده، محتوا ناقص
- **تکمیل**: آماده پیاده‌سازی یا PRD
- **تایید شده**: تایید ذینفعان
