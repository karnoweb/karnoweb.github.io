سیستم رنگ

سیستم رنگ، تایپوگرافی و بصری​

فلسفه طراحی بصری​

طراحی کارنووب باید در نگاه اول حس «SaaS حرفه‌ای» بدهد — نه آژانس طراحی. نه رنگارنگ، نه شلوغ. مدرن، تمیز، قابل اعتماد. مثل Notion، Linear یا Vercel — اما با گرمای کنترل‌شده و هدفمند برای بازار ایران.

اصل کلیدی: رنگ‌های گرمِ لوگو فقط برای «اکشن و تبدیل» استفاده می‌شوند، نه برای کل UI.

سیستم رنگ​

پالت پایه (Core UI / SaaS)​

نامHEXکاربرد
Brand Blue#1A56DBلینک‌ها، حالت فعال، CTAهای عمومی، Highlight
Accent Indigo#4F46E5Hover/Focus، گرادیان سرد، تاکیدهای UI
Dark Navy#0F172Aپس‌زمینه Dark، تیترهای سنگین، بخش‌های Enterprise
Off-White#F8FAFCپس‌زمینه Light، سکشن‌های روشن
Pure White#FFFFFFکارت‌ها، پنل‌ها، Surface اصلی

خنثی‌ها (Neutrals)​

نامHEXکاربرد
Slate 600#475569متن توضیحی، زیرعنوان، متا
Slate 300#CBD5E1Border، Divider، Placeholder
Slate 100#F1F5F9پس‌زمینه سکشن متناوب، کارت Light ثانویه

سیستم Accent برگرفته از لوگو (Logo-derived Accents)​

این رنگ‌ها «هویت لوگو» را وارد UI می‌کنند، اما باید کم‌مصرف و هدفمند باشند.

نامHEXنقش
Action Orange#F16B22CTA اصلی، دکمه اقدام، نقاط تبدیل
Emphasis Red-Orange#E43D30تاکید محدود (Badge/Alert marketing)، نه برای مصرف زیاد
Highlight Yellow#FBD246Highlight/Badge پس‌زمینه، تاکید تصویری
Tech Sky (Cool Accent)#75AEDDAccent سرد برای جزئیات UI، Illustration، Graph
Logo Deep Blue (Optional)#2B4996جایگزین/تکمیل برای پس‌زمینه‌های برند یا گراف‌ها (اختیاری)

نکته: این‌ها رنگ‌های «مبنای طراحی» هستند. اگر در Figma با Eyedropper از فایل لوگو رنگ دقیق‌تری استخراج شد، همین نام‌ها را نگه دارید و فقط HEX را آپدیت کنید تا سیستم شکسته نشود.

تخصیص معنایی (Semantic Mapping)​

برای جلوگیری از سلیقه‌ای شدن، رنگ‌ها باید «معنا» داشته باشند:

  • Primary (UI): Brand Blue
  • Secondary (UI): Accent Indigo
  • Background (Dark): Dark Navy
  • CTA / Conversion: Action Orange (در مارکتینگ سایت)
  • Highlights: Highlight Yellow (ترجیحاً پس‌زمینه Badge، نه متن)
  • Decorative Tech: Tech Sky (برای خطوط، نمودارها، backdropهای ملایم)

قوانین استفاده (Usage Rules)​

  • قاعده 70/20/10

    • 70% خنثی‌ها (Off-White / White / Slateها)
    • 20% رنگ‌های سرد UI (Brand Blue / Accent Indigo / Dark Navy)
    • 10% رنگ‌های لوگو (Orange/Yellow/Red-Orange/Tech Sky)

  • حداکثر ۳ رنگ در هر سکشن (به‌جز خنثی‌ها و خاکستری متن).

  • لینک‌ها همیشه آبی (Brand Blue). اگر لینک نارنجی شود، CTAها قدرت بصری‌شان را از دست می‌دهند.

  • CTA اصلی در سایت مارکتینگ: نارنجی (Action Orange)
    CTA ثانویه: آبی (Outline یا Solid)

  • Dashboard / Client Panel: CTAهای روتین بهتر است آبی باشند؛ نارنجی فقط برای Upsell/Upgrade/Trial/Conversion.

  • نارنجی/قرمز را به‌عنوان رنگ متن روی پس‌زمینه روشن استفاده نکنید مگر با تست کنتراست و اندازه متن مناسب.

Light / Dark Mode (راهنمای سریع)​

Light Mode

  • Background: Off-White / White
  • Text: Dark Navy + Slate 600
  • Primary UI: Brand Blue
  • CTA: Action Orange (کم‌مصرف و مشخص)

Dark Mode

  • Background: Dark Navy + Slate‌های تیره
  • Text: White / نزدیک به White
  • Primary UI: Brand Blue / Accent Indigo
  • CTA: Action Orange + Glow کنترل‌شده

Gradient‌های برند​

/* Cool Gradient (پایه محصول/سرویس) */
--gradient-cool: linear-gradient(135deg, #1A56DB 0%, #4F46E5 100%);

/* Dark Gradient (پس‌زمینه تیره حرفه‌ای) */
--gradient-dark: linear-gradient(135deg, #0F172A 0%, #1E293B 100%);

/* Light Gradient (Surface / Card background) */
--gradient-light: linear-gradient(135deg, #F8FAFC 0%, #EFF6FF 100%);

/* Warm Gradient (CTAهای خیلی مهم / Campaign) */
--gradient-warm: linear-gradient(135deg, #FBD246 0%, #F16B22 45%, #E43D30 100%);

/* Hero Background پیشنهادی (Dark + Glow کنترل‌شده) */
--hero-bg:
  radial-gradient(700px circle at 20% 10%, rgba(241, 107, 34, 0.28), transparent 60%),
  radial-gradient(900px circle at 80% 20%, rgba(26, 86, 219, 0.25), transparent 55%),
  linear-gradient(135deg, #0F172A 0%, #1E293B 100%);

Glow / Shadow سیستم‌مند​

/* Shadow رنگی سرد برای CTA/المان‌های Primary */
--shadow-brand-cool: 0 8px 30px rgba(26, 86, 219, 0.35);

/* Shadow رنگی گرم برای CTAهای Conversion */
--shadow-brand-warm: 0 12px 40px rgba(241, 107, 34, 0.30);

سیستم آیکون​

  • Library اصلی: Heroicons v2 (Outline برای UI، Solid برای تاکید)
  • Size استاندارد: 20px برای UI، 24px برای Feature، 48px برای Section Icon
  • رنگ: از پالت رنگی برند — هرگز رنگ‌های خارج از سیستم نه

قوانین بصری (Do / Don't)​

✓ DO:

  • Dashboard و UI Mockup واقعی در ویژوال‌ها
  • Glassmorphism ملایم روی Dark Background
  • Gradient سرد از Brand Blue به Accent Indigo (در UI)
  • استفاده کنترل‌شده از Warm Gradient فقط برای CTAهای خیلی مهم
  • Floating Cards با Shadow
  • Counter Animation برای اعداد (سبک و سریع)

✗ DON'T:

  • عکس استاک (Stock Photos) به‌جای محتوای واقعی
  • بیش از ۳ رنگ (غیر از خنثی‌ها) در یک سکشن
  • Animation سنگین که سرعت را کم کند
  • Font Size زیر ۱۴px
  • Button بدون Hover/Focus state
  • تبدیل کل سایت به پالت گرم (CTA باید «خاص» بماند)
