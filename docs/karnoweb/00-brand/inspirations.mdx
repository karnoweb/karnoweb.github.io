# الهام‌ها و رفرنس‌ها (Inspirations)

> این سند «کپی‌کردن» نیست؛ یک کتابخانه‌ی مرجع است برای تصمیم‌های UI/Content.
> معیار: Modern SaaS با حس Enterprise، ساده، سریع، و Conversion-oriented.

---

## رفرنس‌های اصلی UI (Core References)

طبق استاندارد برند کارنووب، رفرنس‌های اصلی:

- Vercel
- Linear
- Stripe
- Framer

---

## چرا این رفرنس‌ها؟

### Vercel
- Heroهای تیره و مینیمال
- تایپوگرافی بزرگ + فضای سفید/خالی زیاد
- حس «زیرساخت/پرفورمنس» و اعتماد فنی

### Linear
- نظم در Layout
- فاصله‌گذاری دقیق و آرام
- UI بسیار تمیز، بدون نویز

### Stripe
- ترکیب «سادگی» با «عمق بصری»
- استفاده درست از رنگ‌های Accent بدون شلوغی
- روایت‌سازی مزیت‌ها با مثال‌های ملموس

### Framer
- Motion و تعاملات نرم (Micro-interactions)
- نمایش محصول با حس مدرن
- CTAهای واضح و پرقدرت

---

## الگوهای قابل استفاده (Patterns to Borrow)

### 1) Hero استاندارد کارنووب
- یک تیتر قوی (نتیجه‌محور)
- یک زیرتیتر کوتاه (دامنه راه‌حل)
- دو CTA: Primary (اقدام اصلی) + Secondary (کاهش ریسک)
- Social proof زیر Hero (عدد/لوگو/ویژگی‌های کلیدی)

### 2) Feature Grid
- کارت‌های ساده با آیکون
- ۲ خط توضیح
- یک «نتیجه» یا «وعده قابل سنجش»

### 3) Product Preview
- Screenshot واقعی داشبورد
- Hotspot یا تب‌های ساده (بدون JS سنگین)
- تمرکز روی «کارکرد» نه «تزئین»

### 4) Case Study
- صنعت + مسئله
- راه‌حل
- عدد نتیجه
- CTA: «مشابه شما هستید؟ جلسه بگیرید.»

---

## قواعد استفاده از الهام‌ها

- ساختار بگیر، نه رنگ/فونت/کپی.
- همیشه با توکن‌های `colors-and-theme.md` و تایپوگرافی برند هماهنگ بمان.
- Motion باید به «فهم بهتر» کمک کند، نه فقط نمایش.
