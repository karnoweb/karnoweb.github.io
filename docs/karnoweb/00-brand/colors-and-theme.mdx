## سیستم رنگ، تایپوگرافی و بصری

### فلسفه طراحی بصری

کارنووب باید در نگاه اول حس «SaaS حرفه‌ای» بدهد — نه آژانس طراحی. نه رنگارنگ، نه شلوغ. مدرن، تمیز، قابل اعتماد. مثل Notion، Linear یا Vercel — اما با گرمای مناسب بازار ایران.

### سیستم رنگ

#### رنگ‌های اصلی

| نام | HEX | کاربرد |
|-----|-----|---------|
| **Brand Blue** | `#1A56DB` | CTA اصلی، لینک‌های فعال، Highlight |

#### Gradient‌های برند

```css
/* Gradient اصلی — برای Hero و CTA */
background: linear-gradient(135deg, #1A56DB 0%, #4F46E5 100%);

/* Gradient تیره — برای بخش‌های Dark */
background: linear-gradient(135deg, #0F172A 0%, #1E293B 100%);

/* Gradient روشن — برای کارت‌ها */
background: linear-gradient(135deg, #F8FAFC 0%, #EFF6FF 100%);

/* Gradient Glow — زیر CTA */
box-shadow: 0 0 60px rgba(26, 86, 219, 0.3);
```

### سیستم آیکون

- **Library اصلی:** Heroicons v2 (Outline برای UI، Solid برای تاکید)
- **Size استاندارد:** 20px برای UI، 24px برای Feature، 48px برای Section Icon
- **رنگ:** از پالت رنگی برند — هرگز رنگ‌های خارج از سیستم نه

---

### قوانین بصری (Do / Don't)

**✓ DO:**
- Dashboard و UI Mockup واقعی در ویژوال‌ها
- Glassmorphism ملایم روی Dark Background
- Gradient از Brand Blue به Accent Indigo
- Floating Cards با Shadow
- Counter Animation برای اعداد

**✗ DON'T:**
- عکس استاک (Stock Photos) به‌جای محتوای واقعی
- بیش از ۳ رنگ در یک سکشن
- Animation سنگین که سرعت را کم کند
- Font Size زیر ۱۴px
- Button بدون Hover State
