# Karnoweb Brand & Website Documentation

مستندات ساختاریافته برای طراحی، توسعه و مارکتینگ وب‌سایت **Karnoweb** — قابل توسعه، سازگار با AI، و آماده تبدیل به PRD.

## چرا این مخزن؟

* **قابل توسعه** – هر بخش در پوشه و فایل جداگانه قرار دارد.
* **AI‑Ready** – قالب یکسان در تمام فایل‌ها (نگاه کنید به [docs/TEMPLATE.mdx](docs/TEMPLATE.mdx)).
* **مستند مستقل** – هر سکشن/صفحه فایل خودش را دارد.
* **مسیر PRD** – از مستند بخش‌محور به PRD محصول تبدیل می‌شود.
* **چند تیمی** – برند، UI، فرانت، محتوا، مارکتینگ.
