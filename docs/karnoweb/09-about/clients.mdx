## صفحه مشتریان

### هدف
نمایش دامنه صنایع و کسب‌وکارهایی که به کارنووب اعتماد کرده‌اند + Case Study برای هر دسته.

---

### ساختار کامل صفحه

#### سکشن ۱ — Hero

**تیتر:** کسب‌وکارهایی که با کارنووب رشد کردند

**زیرتیتر:** از فروشگاه‌های آنلاین تا شرکت‌های سازمانی — در صنایع مختلف

**ویژوال:** Mosaic از لوگوهای مشتریان

---

#### سکشن ۲ — لوگو Wall

**تیتر:** برخی از مشتریان ما

**چیدمان:** Grid از لوگوها
- Grayscale عادی → رنگی در Hover
- حداقل ۱۲ لوگو
- Desktop: ۶ در هر ردیف | Mobile: ۳ در هر ردیف

---

#### سکشن ۳ — Case Studies دسته‌بندی‌شده

**تیتر:** نتایجی که اندازه گرفتیم

**Tab‌های فیلتر:** همه | طراحی سایت | SEO | ERP/CRM | اپلیکیشن | اتوماسیون

**ساختار هر Case Study کارت:**

```
┌────────────────────────────────────────────┐
│  [تصویر پروژه یا لوگو مشتری]             │
│                                            │
│  [Tag: صنعت]  [Tag: خدمت]               │
│                                            │
│  نام / صنعت کسب‌وکار                     │
│                                            │
│  مشکل (یک جمله):                         │
│  «موجودی ۳ شعبه به‌صورت دستی مدیریت...» │
│                                            │
│  راهکار:  Shop ERP + اتوماسیون انبار     │
│                                            │
│  نتیجه:                                   │
│  ↑ ۲۸۰٪  دقت موجودی                     │
│  ↓ ۶۰٪   زمان پردازش سفارش             │
│  ↑ ۴۵٪   رضایت مشتریان               │
│                                            │
│  [مشاهده جزئیات کامل →]                 │
└────────────────────────────────────────────┘
```

**تعداد نمایش:** ۶ کارت با «بارگذاری بیشتر»
