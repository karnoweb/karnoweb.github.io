## قیف تبدیل کارنووب (End-to-End)

### هدف سیستم Funnel

**یک اصل:** هیچ ترافیکی نباید «رها» شود.
هر بازدیدکننده‌ای که وارد سایت می‌شود باید یک مسیر مشخص برای تبدیل شدن به مشتری داشته باشد.

---

### Funnel اصلی کارنووب

```
[ورودی: SEO / Ads / Social / Referral]
              ↓
         خانه یا لندینگ
              ↓
    ─────────────────────
    │  Awareness Layer  │
    │  مقاله + محتوا   │
    ─────────────────────
              ↓
    ─────────────────────
    │  Lead Capture     │
    │  PDF / چک‌لیست  │
    │  ورود به CRM     │
    ─────────────────────
              ↓
    ─────────────────────
    │  Consideration    │
    │  Case Study       │
    │  صفحات خدمات    │
    │  دمو محصول      │
    ─────────────────────
              ↓
    ─────────────────────
    │  Conversion       │
    │  جلسه مشاوره    │
    │  درخواست دمو    │
    ─────────────────────
              ↓
    ─────────────────────
    │  Sale             │
    │  پروژه / محصول  │
    ─────────────────────
              ↓
    ─────────────────────
    │  Retention        │
    │  پنل مشتری      │
    │  پشتیبانی       │
    ─────────────────────
              ↓
    ─────────────────────
    │  Upsell           │
    │  ERP / CRM / SEO  │
    │  پشتیبانی ماهانه │
    ─────────────────────
```

---

### ۴ نوع Funnel در کارنووب

**Funnel 1 — Service Funnel (خدمات)**
```
Blog Post (سئو/طراحی/اتوماسیون)
    ↓
صفحه خدمت اختصاصی
    ↓
Case Study مرتبط
    ↓
CTA مشاوره رایگان
    ↓
جلسه → پروژه
```
مثال: «چطور سئو تکنیکال درست انجام دهیم» → `/services/seo` → Case Study → جلسه

---

**Funnel 2 — Product Funnel (محصولات SaaS)**
```
Landing Page محصول
    ↓
Dashboard Preview + Features
    ↓
Pricing Plans
    ↓
درخواست دمو
    ↓
دمو زنده → خرید اشتراک
```

---

**Funnel 3 — Content Funnel (سئو محور)**
```
مقاله آموزشی (ترافیک بلندمدت)
    ↓
Lead Magnet داخل مقاله
    ↓
ورود به Email List / CRM
    ↓
ایمیل Nurture Sequence
    ↓
صفحه خدمت یا محصول
    ↓
Conversion
```

---

**Funnel 4 — Campaign Funnel (تبلیغاتی)**
```
Google Ad / Social Ad
    ↓
Landing Page اختصاصی (نه صفحه اصلی)
    ↓
Offer + فرم ساده
    ↓
تماس از تیم فروش
    ↓
پروژه
```
**نکته مهم برای توسعه:** Landing Pageهای تبلیغاتی **جداگانه** هستند — صفحه اصلی خدمات نباید Destination تبلیغات باشد.

---

### ۷ نقطه احساسی که کاربر باید تجربه کند

```
۱. «این مشکل من را فهمیدند»      ← Pain Point Section
۲. «این‌ها راه‌حل دارند»          ← Solution Section
۳. «قبلاً هم اجرا کرده‌اند»       ← Case Study
۴. «می‌توانم به آن‌ها اعتماد کنم» ← Trust Signals
۵. «بیایم جلسه بگذاریم»           ← CTA اصلی
۶. «شروع کنیم»                    ← Conversion
۷. «بعداً هم می‌توانم توسعه دهم»  ← Upsell Path
```

---

### نقاط شکست Funnel (Critical Drop Points)

**شکست ۱ — Hero Page**
- علت: پیام مبهم یا CTA ضعیف
- راه‌حل: تیتر Outcome-محور + CTA قوی + Trust Bar

**شکست ۲ — صفحه خدمت**
- علت: فقط توضیح خدمت، بدون نتیجه واقعی
- راه‌حل: Case Study با اعداد + FAQ + CTA چندگانه

**شکست ۳ — Case Study**
- علت: داستان بدون عدد، نتیجه مبهم
- راه‌حل: «+۲۸۰٪ ترافیک» نه «ترافیک بهتر شد»

**شکست ۴ — فرم تماس**
- علت: فرم طولانی یا سردرگم‌کننده
- راه‌حل: فرم چندمرحله‌ای، فقط اطلاعات ضروری، پاسخ سریع تضمین شده

### KPIهای Funnel

| مرحله | متریک | هدف |
|-------|--------|------|
| Traffic | Organic Sessions ماهانه | رشد ماهانه |
| Lead Capture | نرخ تبدیل بازدیدکننده به لید | +۳٪ |
| Consultation | نرخ رزرو جلسه از لید | +۳۰٪ |
| Closing | نرخ تبدیل جلسه به پروژه | +۵۰٪ |
| Upsell | درصد مشتریان با محصول ثانویه | +۴۰٪ |
| LTV | ارزش طول عمر مشتری | هدف: ۳x پروژه اول |

### ابزارهای مورد نیاز Funnel

**Marketing Stack:**
- سئو: Ahrefs / Search Console / Screaming Frog
- Analytics: Google Analytics 4 + Hotjar
- Email: MailerLite یا SendGrid

**Sales Stack:**
- CRM: سیستم CRM خود کارنووب
- Booking: Calendly یا سیستم رزرو داخلی
- Proposal: سیستم پروپوزال خودکار

**Retention Stack:**
- پنل مشتری: داخل سیستم
- تیکتینگ: داخلی یا Freshdesk
- گزارش‌دهی: Dashboard اختصاصی
