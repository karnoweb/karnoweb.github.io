## قوانین SEO جهانی سایت

### قوانین فنی (همه صفحات)

**Title Tag:**
- فرمت: `[کلمه کلیدی اصلی] | [توضیح ۲-۳ کلمه] | کارنووب`
- حداکثر ۶۰ کاراکتر
- کلمه کلیدی باید در ابتدای Title باشد

**Meta Description:**
- فرمت: [مشکل یا نیاز] + [راه‌حل کارنووب] + [CTA]
- ۱۴۰-۱۵۵ کاراکتر
- هر صفحه Meta Description منحصربه‌فرد

**Heading Structure:**
- `H1`: فقط یک H1 در هر صفحه — شامل کلمه کلیدی اصلی
- `H2`: بخش‌های اصلی صفحه
- `H3`: زیربخش‌ها
- هرگز H را برای طراحی استفاده نکن

**URL Structure:**
- فارسی نه: `/خدمات/سئو`
- لاتین: `/services/seo`
- خط فاصله (hyphen) نه underscore
- حداکثر ۳ سطح عمق

**Image SEO:**
- `alt` برای همه تصاویر (فارسی + کلمه کلیدی)
- فرمت: WebP
- اندازه: زیر ۱۰۰KB برای تصاویر معمولی
- Lazy Load — به جز تصویر Hero

**Schema Markup:**
- Organization: در همه صفحات (از Header)
- BreadcrumbList: در همه صفحات داخلی
- Service: در صفحات خدمات
- SoftwareApplication: در صفحات محصولات
- FAQPage: هر جا FAQ دارد
- HowTo: صفحه فرآیند
- Person: صفحات تیم

**Core Web Vitals (هدف):**
- LCP: زیر ۲.۵ ثانیه
- FID/INP: زیر ۱۰۰ میلی‌ثانیه
- CLS: زیر ۰.۱
