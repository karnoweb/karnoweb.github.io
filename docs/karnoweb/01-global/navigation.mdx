## هدر، فوتر و سیستم ناوبری

### هدر (Header)

**ساختار:**
```
[Logo کارنووب]  خدمات ▾  محصولات ▾  مقالات  نمونه‌کارها  درباره ما  [تماس با ما] [رزرو جلسه ←]
```

**Dropdown منوی خدمات:**
```
┌──────────────────────────────────────────┐
│ وب و اپلیکیشن    بازاریابی دیجیتال      │
│ ─────────────    ───────────────────     │
│ طراحی سایت       سئو                    │

└──────────────────────────────────────────┘
```

**Dropdown منوی محصولات:**
```
┌────────────────────────────────┐
│ Shop ERP       HR System       │
│ CRM            LMS             │
│ حسابداری      Club Management  │
│                                │
│ [مشاهده همه محصولات ←]       │

└────────────────────────────────┘
```

**دکمه‌های راست هدر:**
- «تماس با ما» — لینک ساده، رنگ متن
- «رزرو جلسه رایگان» — Primary Button

**Sticky Header:**
- Scroll زیر ۸۰px: شفاف با Blur
- Scroll بیش از ۸۰px: پس‌زمینه سفید + Shadow

**Mobile Header:**
- Hamburger Menu با Slide-in از راست
- تمام آیتم‌ها Accordion
