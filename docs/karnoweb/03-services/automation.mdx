## صفحه اتوماسیون کسب‌وکار

### پیام اصلی
> «زمانی که صرف کارهای تکراری می‌کنید، از رشد کسب‌وکارتان می‌زنید.»

---

### ساختار کامل صفحه

#### سکشن ۱ — Hero

| عنصر | محتوا |
|------|--------|
| **تیتر** | فرآیندهای کسب‌وکارت را هوشمند کن — وقتت برای رشد است |
| **زیرتیتر** | اتوماسیون فرآیندها، حذف کارهای تکراری و کاهش خطای انسانی با سیستم‌های هوشمند |
| **CTA اول** | تحلیل رایگان فرآیندها → `/contact?service=automation&offer=analysis` |
| **ویژوال** | Workflow Diagram از یک اتوماسیون واقعی |

---

#### سکشن ۲ — چه چیزهایی را اتوماتیک می‌کنیم؟

| حوزه | مثال‌های واقعی |
|------|----------------|
| **فروش** | ارسال خودکار پیامک تایید سفارش، پیگیری فاکتور پرداخت‌نشده |
| **CRM** | ورود خودکار لیدها از سایت به CRM، تخصیص لید به فروشنده |
| **انبار** | هشدار کمبود موجودی، سفارش خودکار از تامین‌کننده |
| **مالی** | صدور فاکتور خودکار، گزارش روزانه مالی |
| **HR** | ثبت خودکار حضور، محاسبه حقوق، اعلان مرخصی |
| **بازاریابی** | Email Drip Campaign، پیگیری لید سرد |

---

#### سکشن ۳ — تکنولوژی‌ها

| ابزار | کاربرد |
|-------|--------|
| Laravel Jobs & Queues | پردازش‌های پس‌زمینه پایدار |
| Webhook | اتصال بین سرویس‌های مختلف |
| n8n / Zapier | اتوماسیون‌های No-Code |
| API Gateway | یکپارچه‌سازی سیستم‌های مختلف |
| AI Integration | پردازش هوشمند داده‌ها |

---

#### سکشن ۴ — فرآیند اجرا

| مرحله | کار |
|-------|-----|
| ۱. Map فرآیند | رسم دقیق فرآیند فعلی و شناسایی گلوگاه‌ها |
| ۲. طراحی اتوماسیون | تعریف ورودی، خروجی و شرایط هر قانون |
| ۳. توسعه | پیاده‌سازی + تست |
| ۴. آموزش | آموزش تیم مشتری |
| ۵. پایش | نظارت بر عملکرد + بهینه‌سازی |

| CTA | متن | لینک |

### ارتباط با صفحات دیگر
- `←` از: `/services`، صفحات محصولات ERP و CRM
- `→` به: `/contact`، `/products/shop-erp`، `/products/crm`
- **Upsell:** «اتوماسیون + ERP = حذف کامل کارهای دستی» → `/products/shop-erp`
