## صفحه اصلی خدمات (صفحه مادر)

### هدف صفحه
معرفی کامل خدمات کارنووب به کاربری که نمی‌داند دقیقاً چه نیازی دارد یا از منوی ناوبری وارد شده.

### اهداف تبدیل
۱. هدایت کاربر به صفحه خدمت مرتبط
۲. جذب لید از طریق فرم مشاوره اولیه
۳. نمایش تخصص جامع و ایجاد اعتماد

### پیام اصلی
> «کارنووب فقط طراح سایت نیست — شریک رشد دیجیتال کسب‌وکار شماست.»

---

### ساختار کامل صفحه

#### سکشن ۱ — Hero

| عنصر | محتوا |
|------|--------|
| **تیتر اصلی** | راهکارهای دیجیتال برای رشد واقعی کسب‌وکار شما |
| **زیرتیتر** | از طراحی سایت و اپلیکیشن تا سئو، اتوماسیون و سامانه‌های سازمانی — همه‌چیز زیر یک سقف |
| **CTA اول** | دریافت مشاوره رایگان → لینک به `/contact?source=services-hero` |
| **CTA دوم** | مشاهده نمونه‌کارها → لینک به `/portfolio` |
| **ویژوال** | گرید کارت‌های خدمات با آیکون‌های انیمیشنی — نه عکس استاک |

**راهنمای طراحی Hero:**
- پس‌زمینه: گرادیان تیره (از رنگ اصلی برند به سمت سیاه ملایم)
- تیتر: فونت بزرگ، وزن Bold، رنگ سفید
- زیر Hero: نوار اعتمادسازی شامل `+۱۲۰ پروژه | +۴۰ مشتری فعال | ۹ سال تجربه`

---

#### سکشن ۲ — دسته‌بندی خدمات (فیلتر تعاملی)

**هدف:** کاربر سریع خدمت مرتبط با کسب‌وکارش را پیدا کند.

**Tab های فیلتر:**
- همه خدمات
- وب و اپلیکیشن
- بازاریابی دیجیتال
- سامانه‌های سازمانی
- طراحی و برند

**ساختار هر کارت خدمت:**
```
┌─────────────────────────────────┐
│  [آیکون متمایز — نه generic]   │
│                                 │
│  عنوان خدمت                    │
│  توضیح ۲ خطی — نتیجه‌محور     │
│                                 │
│  ✓ ویژگی کلیدی ۱              │
│  ✓ ویژگی کلیدی ۲              │
│                                 │
│  [مشاهده جزئیات ←]            │
└─────────────────────────────────┘
```

**کارت‌های خدمات و لینک‌ها:**

| خدمت | زیرعنوان | نتیجه | لینک |
|------|-----------|--------|------|
| سئو و رشد ارگانیک | افزایش ورودی هدفمند گوگل | افزایش فروش و لید | `/services/seo` |
| طراحی سایت اختصاصی | وبسایت سریع، مدرن، مقیاس‌پذیر | افزایش اعتماد و Conversion | `/services/website` |
| فروشگاه اینترنتی | موتور فروش آنلاین حرفه‌ای | افزایش درآمد آنلاین | `/services/ecommerce` |
| اپلیکیشن موبایل | iOS و Android با Backend قوی | حضور در دستان کاربر | `/services/app` |
| UI/UX | طراحی تجربه‌محور | کاهش خروج، افزایش تبدیل | `/services/uiux` |
| تولید محتوا | محتوای استراتژیک | ترافیک + اعتماد + لید | `/services/content` |
| برندینگ | هویت بصری کامل | برند قابل اعتماد | `/services/branding` |
| گوگل ادز | کمپین هدفمند با ROI واقعی | فروش فوری | `/services/ads` |
| اتوماسیون | هوشمندسازی فرآیندها | صرفه‌جویی زمان و هزینه | `/services/automation` |
| مشاوره استارتاپ | از ایده تا اجرا | مسیر درست از ابتدا | `/services/consulting` |

**UI Direction:**
- Desktop: Grid 3 ستونه
- Tablet: Grid 2 ستونه
- Mobile: Stack کارت‌ها با Horizontal Scroll برای Tab‌ها
- Hover: Elevation + Border Glow با رنگ برند
- Animation: کارت‌ها با Stagger Fade-in هنگام Scroll

#### سکشن ۳ — چرا کارنووب؟ (مزیت رقابتی)

**هدف:** پاسخ به سوال ذهنی «چرا اینجا و نه جای دیگه؟»

**ساختار نمایش:** Grid 2 ستونه با آیکون‌های بزرگ

| مزیت | توضیح |
|------|--------|
| **معماری مقیاس‌پذیر** | سیستم‌هایی که با رشد کسب‌وکار شما بزرگ می‌شوند |
| **API-First Development** | اتصال آسان به هر سرویس خارجی |
| **SEO Technical** | ساختار فنی بهینه از روز اول |
| **Conversion-Focused** | طراحی برای فروش، نه فقط زیبایی |
| **DevOps & CI/CD** | استقرار سریع و پایدار |
| **پشتیبانی واقعی** | نه فقط پس از تحویل — همیشه کنار شماییم |

#### سکشن ۴ — فرآیند همکاری (خلاصه)

**هدف:** کاهش ترس از شروع همکاری

**مراحل (۴ مرحله ساده):**
```
[۱ تحلیل]  →  [۲ طراحی]  →  [۳ توسعه]  →  [۴ رشد]
```
زیر هر مرحله: یک جمله توضیح

#### سکشن ۵ — نمونه‌کارها (Preview)

**تعداد:** ۳ پروژه برتر (انتخاب از متنوع‌ترین خدمات)
**ساختار هر کارت:**
- تصویر پروژه
- نام مشتری / صنعت
- خدمات ارائه شده: Tag‌های رنگی
- نتیجه: `«+۱۸۰٪ افزایش ورودی گوگل»`
- لینک: مشاهده جزئیات → `/portfolio/[slug]`

**CTA پایان سکشن:** مشاهده همه نمونه‌کارها → `/portfolio`

---

#### سکشن ۶ — Lead Magnet (جذب لید نرم)

**هدف:** کاربری که هنوز آماده مشاوره نیست را به Funnel وارد کنیم.

**پیشنهاد:**
> «نمی‌دانی از کجا شروع کنی؟ چک‌لیست رشد دیجیتال کسب‌وکار را رایگان دریافت کن.»

**فرم:** نام + ایمیل + دکمه «دریافت رایگان»
**بعد از ثبت:** ارسال فایل PDF + ورود به CRM برای پیگیری

---

#### سکشن ۷ — CTA نهایی

**Title:** آماده‌ای رشد دیجیتال کسب‌وکارت را شروع کنی؟
**زیرعنوان:** اولین جلسه مشاوره رایگان است. بدون تعهد.

| CTA | متن | لینک |
|-----|-----|------|
| Primary | رزرو جلسه مشاوره | `/contact?type=consultation` |
| Secondary | دریافت پیشنهاد قیمت | `/contact?type=quote` |

**عناصر اعتمادسازی زیر CTAها:**
- ✓ پاسخگویی در کمتر از ۲۴ ساعت
- ✓ مشاوره تخصصی بدون فروش اجباری
- ✓ تحلیل اولیه رایگان

### SEO صفحه
- **Slug:** `/services`
- **Title Tag:** خدمات دیجیتال | طراحی سایت، سئو، اپلیکیشن | کارنووب
- **Meta Description:** کارنووب ارائه‌دهنده خدمات طراحی سایت، سئو، اپلیکیشن موبایل، ERP و اتوماسیون کسب‌وکار. مشاوره رایگان بگیرید.
- **Schema:** Service Schema برای هر خدمت

### ارتباط با صفحات دیگر
- `←` از: منوی اصلی، Hero خانه، Footer
- `→` به: هر صفحه خدمت اختصاصی، `/portfolio`، `/contact`
- داخلی: لینک به `/about` در سکشن مزیت رقابتی

## راهنمای پیاده‌سازی عمومی (برای توسعه‌دهنده)

### URL Structure
```
/services                          ← صفحه مادر
/services/seo                      ← سئو
/services/website                  ← طراحی سایت
/services/ecommerce                ← فروشگاه
/services/app                      ← اپلیکیشن
/services/uiux                     ← UI/UX
/services/content                  ← تولید محتوا
/services/branding                 ← برندینگ
/services/ads                      ← گوگل ادز
/services/automation               ← اتوماسیون
/services/consulting               ← مشاوره استارتاپ
```

### Query String برای تحلیل
همه فرم‌های تماس باید `?service=` و `?source=` را به CRM ارسال کنند تا مشخص باشد لید از کجا آمده.

### Breadcrumb همه صفحات خدمات
```
خانه > خدمات > [نام خدمت]
```

### Navigation داخل خدمات
در پایین هر صفحه خدمت، یک بخش «خدمات مرتبط» با ۲-۳ کارت از صفحات مشابه قرار بده.

### Mobile-First Rules
- CTA ها Sticky در پایین صفحه موبایل
- فونت‌ها حداقل ۱۶px
- Tap Target حداقل ۴۸px
- Accordion برای FAQ و لیست‌های طولانی

### Performance
- هر صفحه خدمات: LCP زیر ۲.۵ ثانیه
- تصاویر: WebP + Lazy Load
- Hero: LCP Critical — No Lazy Load
