صفحه اصلی خدمات (صفحه مادر)​

هدف صفحه​

معرفی کامل خدمات کارنووب به کاربری که نمی‌داند دقیقاً چه نیازی دارد یا از منوی ناوبری وارد شده.

اهداف تبدیل​

۱. هدایت کاربر به صفحه خدمت مرتبط ۲. جذب لید از طریق فرم مشاوره اولیه ۳. نمایش تخصص جامع و ایجاد اعتماد

پیام اصلی​

«کارنووب فقط طراح سایت نیست — شریک رشد دیجیتال کسب‌وکار شماست.»

ساختار کامل صفحه​

سکشن ۱ — Hero​

عنصرمحتوا
تیتر اصلیراهکارهای دیجیتال برای رشد واقعی کسب‌وکار شما
زیرتیتراز طراحی سایت و اپلیکیشن تا سئو، اتوماسیون و سامانه‌های سازمانی — همه‌چیز زیر یک سقف
CTA اولدریافت مشاوره رایگان → لینک به /contact?source=services-hero
CTA دوممشاهده نمونه‌کارها → لینک به /portfolio
ویژوالگرید کارت‌های خدمات با آیکون‌های انیمیشنی — نه عکس استاک

راهنمای طراحی Hero:

  • پس‌زمینه: گرادیان تیره (از رنگ اصلی برند به سمت سیاه ملایم)
  • تیتر: فونت بزرگ، وزن Bold، رنگ سفید
  • زیر Hero: نوار اعتمادسازی شامل +۱۲۰ پروژه | +۴۰ مشتری فعال | ۹ سال تجربه

سکشن ۲ — دسته‌بندی خدمات (فیلتر تعاملی)​

هدف: کاربر سریع خدمت مرتبط با کسب‌وکارش را پیدا کند.

Tab های فیلتر:

  • همه خدمات
  • وب و اپلیکیشن
  • بازاریابی دیجیتال
  • سامانه‌های سازمانی
  • طراحی و برند

ساختار هر کارت خدمت:

┌─────────────────────────────────┐
│  [آیکون متمایز — نه generic]   │
│                                 │
│  عنوان خدمت                    │
│  توضیح ۲ خطی — نتیجه‌محور     │
│                                 │
│  ✓ ویژگی کلیدی ۱              │
│  ✓ ویژگی کلیدی ۲              │
│                                 │
│  [مشاهده جزئیات ←]            │
└─────────────────────────────────┘

کارت‌های خدمات و لینک‌ها:

خدمتزیرعنواننتیجهلینک
سئو و رشد ارگانیکافزایش ورودی هدفمند گوگلافزایش فروش و لید/services/seo
طراحی سایت اختصاصیوبسایت سریع، مدرن، مقیاس‌پذیرافزایش اعتماد و Conversion/services/website
فروشگاه اینترنتیموتور فروش آنلاین حرفه‌ایافزایش درآمد آنلاین/services/ecommerce
اپلیکیشن موبایلiOS و Android با Backend قویحضور در دستان کاربر/services/app
UI/UXطراحی تجربه‌محورکاهش خروج، افزایش تبدیل/services/uiux
تولید محتوامحتوای استراتژیکترافیک + اعتماد + لید/services/content
برندینگهویت بصری کاملبرند قابل اعتماد/services/branding
گوگل ادزکمپین هدفمند با ROI واقعیفروش فوری/services/ads
اتوماسیونهوشمندسازی فرآیندهاصرفه‌جویی زمان و هزینه/services/automation
مشاوره استارتاپاز ایده تا اجرامسیر درست از ابتدا/services/consulting

UI Direction:

  • Desktop: Grid 3 ستونه
  • Tablet: Grid 2 ستونه
  • Mobile: Stack کارت‌ها با Horizontal Scroll برای Tab‌ها
  • Hover: Elevation + Border Glow با رنگ برند
  • Animation: کارت‌ها با Stagger Fade-in هنگام Scroll

سکشن ۳ — چرا کارنووب؟ (مزیت رقابتی)​

هدف: پاسخ به سوال ذهنی «چرا اینجا و نه جای دیگه؟»

ساختار نمایش: Grid 2 ستونه با آیکون‌های بزرگ

مزیتتوضیح
معماری مقیاس‌پذیرسیستم‌هایی که با رشد کسب‌وکار شما بزرگ می‌شوند
API-First Developmentاتصال آسان به هر سرویس خارجی
SEO Technicalساختار فنی بهینه از روز اول
Conversion-Focusedطراحی برای فروش، نه فقط زیبایی
DevOps & CI/CDاستقرار سریع و پایدار
پشتیبانی واقعینه فقط پس از تحویل — همیشه کنار شماییم

سکشن ۴ — فرآیند همکاری (خلاصه)​

هدف: کاهش ترس از شروع همکاری

مراحل (۴ مرحله ساده):

[۱ تحلیل]  →  [۲ طراحی]  →  [۳ توسعه]  →  [۴ رشد]

زیر هر مرحله: یک جمله توضیح CTA: «مراحل کامل را ببینید» → لینک به بخش فرآیند صفحه درباره ما

سکشن ۵ — نمونه‌کارها (Preview)​

تعداد: ۳ پروژه برتر (انتخاب از متنوع‌ترین خدمات) ساختار هر کارت:

  • تصویر پروژه
  • نام مشتری / صنعت
  • خدمات ارائه شده: Tag‌های رنگی
  • نتیجه: «+۱۸۰٪ افزایش ورودی گوگل»
  • لینک: مشاهده جزئیات → /portfolio/[slug]

CTA پایان سکشن: مشاهده همه نمونه‌کارها → /portfolio

سکشن ۶ — Lead Magnet (جذب لید نرم)​

هدف: کاربری که هنوز آماده مشاوره نیست را به Funnel وارد کنیم.

پیشنهاد:

«نمی‌دانی از کجا شروع کنی؟ چک‌لیست رشد دیجیتال کسب‌وکار را رایگان دریافت کن.»

فرم: نام + ایمیل + دکمه «دریافت رایگان» بعد از ثبت: ارسال فایل PDF + ورود به CRM برای پیگیری

سکشن ۷ — CTA نهایی​

Title: آماده‌ای رشد دیجیتال کسب‌وکارت را شروع کنی؟ زیرعنوان: اولین جلسه مشاوره رایگان است. بدون تعهد.

CTAمتنلینک
Primaryرزرو جلسه مشاوره/contact?type=consultation
Secondaryدریافت پیشنهاد قیمت/contact?type=quote

عناصر اعتمادسازی زیر CTAها:

  • ✓ پاسخگویی در کمتر از ۲۴ ساعت
  • ✓ مشاوره تخصصی بدون فروش اجباری
  • ✓ تحلیل اولیه رایگان

SEO صفحه​

  • Slug: /services
  • Title Tag: خدمات دیجیتال | طراحی سایت، سئو، اپلیکیشن | کارنووب
  • Meta Description: کارنووب ارائه‌دهنده خدمات طراحی سایت، سئو، اپلیکیشن موبایل، ERP و اتوماسیون کسب‌وکار. مشاوره رایگان بگیرید.
  • Schema: Service Schema برای هر خدمت

ارتباط با صفحات دیگر​

  • ← از: منوی اصلی، Hero خانه، Footer
  • → به: هر صفحه خدمت اختصاصی، /portfolio، /contact
  • داخلی: لینک به /about در سکشن مزیت رقابتی

راهنمای پیاده‌سازی عمومی (برای توسعه‌دهنده)​

URL Structure​

/services                          ← صفحه مادر
/services/seo                      ← سئو
/services/website                  ← طراحی سایت
/services/ecommerce                ← فروشگاه
/services/app                      ← اپلیکیشن
/services/uiux                     ← UI/UX
/services/content                  ← تولید محتوا
/services/branding                 ← برندینگ
/services/ads                      ← گوگل ادز
/services/automation               ← اتوماسیون
/services/consulting               ← مشاوره استارتاپ

Query String برای تحلیل​

همه فرم‌های تماس باید ?service= و ?source= را به CRM ارسال کنند تا مشخص باشد لید از کجا آمده.

Breadcrumb همه صفحات خدمات​

خانه > خدمات > [نام خدمت]

Navigation داخل خدمات​

در پایین هر صفحه خدمت، یک بخش «خدمات مرتبط» با ۲-۳ کارت از صفحات مشابه قرار بده.

Mobile-First Rules​

  • CTA ها Sticky در پایین صفحه موبایل
  • فونت‌ها حداقل ۱۶px
  • Tap Target حداقل ۴۸px
  • Accordion برای FAQ و لیست‌های طولانی

Performance​

  • هر صفحه خدمات: LCP زیر ۲.۵ ثانیه
  • تصاویر: WebP + Lazy Load
  • Hero: LCP Critical — No Lazy Load
