# راهنمای سئو بلاگ (Blog SEO Guidelines)


## هدف

- تضمین اینکه همه مقالات استاندارد سئو را رعایت کنند
- کمک به تیم نگارش برای تولید محتوای سئو-محور
- کاهش اشتباهات رایج که ترافیک ارگانیک را خراب می‌کند

---

## Keyword Strategy (استراتژی کلیدواژه)

### ۳ نوع کلیدواژه در هر مقاله

1) **Primary Keyword (کلیدواژه اصلی)**
   - یک عدد
   - Volume متوسط تا بالا / Difficulty متناسب با DA سایت
   - باید در: H1، مقدمه، Meta Title، Meta Description، URL باشد

2) **Secondary Keywords (کلیدواژه‌های مرتبط)**
   - ۳–۵ عدد
   - کلیدواژه‌های مرتبط و LSI
   - در H2ها و بدنه مقاله به‌طور طبیعی استفاده شوند

3) **Long-tail Keywords (کلیدواژه‌های دنباله‌دار)**
   - سوالاتی که کاربر می‌پرسد (People Also Ask)
   - در بخش FAQ مقاله پوشش داده شوند

---

## On-page SEO Checklist (برای هر مقاله)

### Title Tag
- [ ] حاوی Primary Keyword
- [ ] حداکثر ۶۰ کاراکتر
- [ ] جذاب و کلیک‌پذیر
- [ ] Keyword نزدیک ابتدا
- [ ] فرمت: `[Keyword] — [توضیح] | کارنووب`

### Meta Description
- [ ] ۱۴۰–۱۵۵ کاراکتر
- [ ] حاوی Primary Keyword
- [ ] فرمت: مشکل + راه‌حل + CTA
- [ ] برای هر مقاله یکتا

### URL
- [ ] انگلیسی، lowercase، با hyphen
- [ ] حاوی Primary Keyword (انگلیسی)
- [ ] کوتاه (حداکثر ۵–۶ کلمه)
- [ ] ساختار: `/blog/[category]/[article-slug]`
- [ ] بدون stop words در URL

### Heading Structure
- [ ] H1: یک عدد، حاوی Primary Keyword
- [ ] H2ها: بخش‌های اصلی، حاوی Secondary Keywords
- [ ] H3ها: زیربخش‌ها
- [ ] از H4 به پایین جز در موارد ضروری استفاده نشود

### محتوا
- [ ] Primary Keyword در ۱۰۰ کلمه اول
- [ ] Keyword Density طبیعی (نه Stuffing)
- [ ] حداقل ۱ لینک خارجی معتبر (اگر لازم است)
- [ ] حداقل ۲ لینک داخلی به خدمات/محصولات/مقالات
- [ ] Anchor Text توصیفی (نه «کلیک کنید» یا «اینجا»)

### تصاویر
- [ ] فرمت WebP
- [ ] Alt text برای همه تصاویر (حاوی Keyword اگر مرتبط است)
- [ ] Lazy-load (به‌جز Featured Image)
- [ ] اندازه: حجم کمتر از ۱۵۰KB برای تصاویر معمولی

### Schema
- [ ] `Article` یا `BlogPosting`
- [ ] `BreadcrumbList`
- [ ] `FAQPage` (اگر بخش FAQ وجود دارد)
- [ ] `HowTo` (اگر مقاله گام‌به‌گام است)

---

## Internal Linking Strategy (لینک‌سازی داخلی)

### قوانین
- هر مقاله حداقل ۲–۳ لینک داخلی داشته باشد
- لینک به صفحات اصلی خدمات/محصولات از مقالات مرتبط
- لینک بین مقالات یک Topic Cluster
- Anchor Text توصیفی و طبیعی (نه کلیدواژه مصنوعی)
- صفحات پر اهمیت (Services/Products) باید لینک داخلی بیشتری دریافت کنند

### مسیرهای رایج لینک داخلی
- مقاله سئو → `/services/seo`
- مقاله ERP → `/products/erp`
- مقاله CRM → `/products/crm`
- مقاله اتوماسیون → `/services/automation`

---

## Content Freshness (به‌روزرسانی محتوا)

### برنامه به‌روزرسانی
- Evergreen Posts: هر ۶ ماه بررسی
- Authority Posts: هر ۱۲ ماه بررسی
- Conversion Posts: هر ۳ ماه بررسی

### چه چیزی به‌روزرسانی شود؟
- اعداد و آمارها
- ابزارها و تکنولوژی‌ها
- لینک‌های شکسته
- CTAها (اگر خدمت/محصول تغییر کرده)
- تاریخ آخرین به‌روزرسانی در Schema و صفحه

---

## Topic Cluster (خوشه موضوعی)

### اصل
- هر دسته بلاگ یک «Pillar Page» دارد
- Pillar Page کلی‌ترین موضوع را هدف می‌گیرد
- Cluster Pages (مقالات زیرمجموعه) موضوعات جزئی‌تر را پوشش می‌دهند
- Pillar ↔ Cluster لینک دوطرفه دارند

### مثال
- Pillar: «راهنمای جامع ERP»
- Cluster: «چطور ERP را انتخاب کنیم؟»
- Cluster: «اشتباهات رایج در پیاده‌سازی ERP»
- Cluster: «فازبندی ERP برای شرکت‌های متوسط»

---

## Core Web Vitals (اهداف اجرایی)

- LCP: زیر ۲.۵ ثانیه
- INP: زیر ۱۰۰ میلی‌ثانیه
- CLS: زیر ۰.۱

### چک‌لیست عملکرد مقالات
- [ ] Featured Image ابعاد مشخص دارد (بدون Layout Shift)
- [ ] فونت‌ها Preload شده‌اند
- [ ] JS غیرضروری در مقالات نیست
- [ ] Lazy-load درست پیاده شده

---

## اشتباهات رایج سئو بلاگ (ممنوع)

| اشتباه | دلیل | جایگزین |
|-------|------|---------|
| Keyword Stuffing | جریمه گوگل | استفاده طبیعی |
| Duplicate Content | افت رتبه | Canonical یا بازنویسی |
| H1 تکراری | گیج کردن خزنده | یک H1 در هر صفحه |
| Anchor Text «اینجا» | ضعیف | Anchor توصیفی |
| تصویر بدون Alt | افت دسترس‌پذیری + سئو | Alt توصیفی |
| URL فارسی | مشکل در اشتراک‌گذاری | Slug انگلیسی |
| Meta Description تکراری | گیج شدن Google | یکتا برای هر صفحه |

## ابزارهای پیشنهادی

- Keyword Research: Google Search Console, Ahrefs, Google Trends
- On-page: Screaming Frog, Ahrefs Site Audit
- عملکرد: PageSpeed Insights, Google Search Console
- Schema Test: Google Rich Results Test
