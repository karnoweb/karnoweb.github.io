## سکشن پیش‌نمایش محصولات

### هدف
نمایش اینکه کارنووب **فقط شرکت خدماتی نیست** — محصولات نرم‌افزاری واقعی ساخته و نگهداری می‌کند. این سکشن اعتبار فنی را چند برابر می‌کند.

### پیام اصلی
> «سامانه‌های آماده برای رشد سریع‌تر کسب‌وکارها — نه پروژه سفارشی، بلکه محصول آماده با دمو فوری.»

---

### ساختار سکشن

**Header:**

| عنصر | محتوا |
|------|--------|
| **Eyebrow** | محصولات نرم‌افزاری |
| **Title** | سیستم‌های آماده و مقیاس‌پذیر |
| **Subtitle** | ERP، CRM، حسابداری و سامانه‌های سازمانی — آماده راه‌اندازی برای کسب‌وکار شما |

---

**ساختار کارت محصول:**
```
┌──────────────────────────────────────┐
│  [Screenshot داشبورد واقعی]         │
```

**محصولات:**

| محصول | توضیح | ویژگی‌های کلیدی | لینک |
|-------|--------|-----------------|------|
| Shop ERP | مدیریت فروش، انبار، سفارشات و عملیات فروشگاهی | Multi-Branch · API · گزارش لحظه‌ای | `/products/shop-erp` |
| CRM | مدیریت ارتباط با مشتریان و فرآیند فروش | Pipeline · یادآور · پیامک | `/products/crm` |
| حسابداری | سیستم حسابداری و گزارشات مالی حرفه‌ای | فاکتور · مدیریت چک · Cloud | `/products/accounting` |
| HR System | مدیریت منابع انسانی، حضور و حقوق | حقوق خودکار · مرخصی آنلاین | `/products/hr-system` |
| LMS | پلتفرم آموزش آنلاین و مدیریت دانشجو | ویدیو امن · گواهی · فروش دوره | `/products/lms` |
| Club Management | مدیریت باشگاه‌ها و فرآیندهای عضویت | اشتراک · کنترل ورود · یادآور | `/products/club-management` |

**UI Direction:**
- Desktop: Grid 3 ستونه (2 ردیف)
- Interactive: Tab‌های دسته‌بندی (SaaS / Enterprise)
- هر کارت: Screenshot واقعی، نه Mockup طراحی‌شده
- Hover: Overlay با CTA «درخواست دمو»
- Badge «جدید» یا «پرفروش» در گوشه کارت‌های مناسب
