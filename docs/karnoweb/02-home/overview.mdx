# صفحه خانه (Home) — نمای کلی

> وضعیت: پیش‌نویس  
> بخش: 02-home  
> آخرین به‌روزرسانی: 2026-07-01

## Goal (هدف)

صفحه خانه باید در کمترین زمان ممکن:
- اعتماد بسازد (Enterprise-ready)
- کارنووب را شفاف معرفی کند (سیستم رشد + زیرساخت دیجیتال، نه صرفاً طراحی سایت)
- خدمات و محصولات را «نتیجه‌محور» نشان دهد
- کاربر را وارد قیف تبدیل کند (رزرو جلسه / درخواست دمو / ثبت لید)

## Target Audience (مخاطب هدف)

- صاحبان کسب‌وکار (B2B و B2C با نیاز رشد و سیستم‌سازی)
- مدیران مارکتینگ / فروش / عملیات
- مدیر IT / CTO
- استارتاپ‌ها (در فاز ساخت MVP تا Scale)

## User Problems (مسئله‌های کاربر)

- «این تیم دقیقاً چیکار می‌کند؟ آژانس است یا شرکت محصول؟»
- «آیا واقعاً توان فنی + درک بیزنسی دارد؟»
- «اگر شروع کنم، مسیر و ریسک‌ها چیست؟»
- «نمونه نتیجه واقعی دارید یا فقط ادعاست؟»
- «قدم بعدی من چیست؟ تماس؟ دمو؟ مشاوره؟»

## Solutions (راه‌حل‌هایی که صفحه ارائه می‌دهد)

- پیام یک‌خطی واضح در Hero + CTA مشخص
- Social Proof سریع (لوگو/عدد/نقل‌قول کوتاه)
- پیش‌نمایش خدمات با تمرکز روی «مسئله→راه‌حل→نتیجه»
- پیش‌نمایش محصولات با «اسکرین‌شات واقعی» + CTA دمو
- سکشن «چرا کارنووب؟» با تمایزهای قابل دفاع
- فرآیند همکاری شفاف (کاهش ترس)
- Case Study با عدد (اثبات)
- مقالات (اعتبار + سئو)
- Lead Magnet (تبدیل کاربر مردد به لید)
- Final CTA (قوی‌ترین دعوت به اقدام)

## CTA (دعوت به اقدام)

### CTA اصلی صفحه (Primary)
- «رزرو جلسه استراتژی / مشاوره رایگان»
- مقصد پیشنهادی: `/contact?type=consultation&source=home`

> قانون رنگ: چون صفحه «مارکتینگ» است، Primary باید Warm باشد (طبق `01-global/cta-strategy.md` و `00-brand/colors-and-theme.md`).

### CTAهای ثانویه (Secondary)
- «درخواست دمو» (برای کاربر محصول‌محور)
- «مشاهده نمونه‌کارها»
- «مشاهده خدمات / محصولات»

### CTA نرم (Soft)
- «دریافت چک‌لیست / دانلود PDF»
- «تحلیل رایگان سایت»

## SEO Keywords (کلیدواژه‌های سئو)

> هدف: ترکیب کلیدواژه‌های «خدمت» + «سیستم‌های سازمانی» + «اتوماسیون/زیرساخت»

- طراحی سایت اختصاصی
- سئو تکنیکال / خدمات سئو
- اتوماسیون کسب‌وکار
- طراحی و پیاده‌سازی ERP
- طراحی و پیاده‌سازی CRM
- توسعه نرم‌افزار سفارشی (در صورت استراتژی محتوا)
- زیرساخت رشد دیجیتال
- مشاوره رشد دیجیتال

## UI Ideas (ایده‌های UI)

- حس کلی: Modern SaaS / Enterprise (مرجع: Vercel/Linear/Stripe/Framer)
- رنگ: پایه Cool + Accent Warm فقط برای Conversion
- Hero: پس‌زمینه Dark با `--hero-bg` (Glow کنترل‌شده)
- نمایش محصول: Screenshot واقعی + کارت‌های شناور
- تایپوگرافی: تیتر بزرگ و کوتاه، متن کم و اسکن‌پذیر
- CTA sticky در Header (طبق `01-global/header.md`)

## Conversion Strategy (استراتژی تبدیل)

- صفحه خانه باید هم‌زمان دو مسیر را پوشش دهد:
  1) مسیر خدمات: مقاله/نمونه‌کار → رزرو جلسه
  2) مسیر محصول: محصول → درخواست دمو

- هر سکشن باید «یک قدم بعدی» داشته باشد؛ CTA زیاد ممنوع.
- Lead Magnet برای کاربرانی که آماده خرید نیستند (کاهش خروج).

## Notes (یادداشت‌ها)

### متادیتا (پیشنهادی)
- URL: `/`
- Title Tag پیشنهادی: «کارنووب | زیرساخت دیجیتال، سئو، اتوماسیون و سامانه‌های سازمانی»
- Meta Description پیشنهادی: «کارنووب شریک رشد دیجیتال کسب‌وکارهاست؛ از سئو و توسعه وب‌سایت تا ERP/CRM و اتوماسیون. جلسه مشاوره رایگان رزرو کنید.»

### Schema (پیشنهادی)
- Organization + WebSite + SiteLinksSearchBox

- LCP: زیر 2.5 ثانیه (عنصر LCP در Hero lazy-load نشود)
- INP: زیر 100ms
